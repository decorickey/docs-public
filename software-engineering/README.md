## [オブジェクト指向UI vs タスク指向](https://gihyo.jp/book/2020/978-4-297-11351-3)

|オブジェクト指向|タスク指向|
|-|-|
|GUI|CUI|
|名刺→動詞|動詞→名刺|
|モードレス（モードがない）|モーダル（モードがある）|
|日常の動作と同じ自然な流れ|日常の動作と異なる不自然な流れ|
