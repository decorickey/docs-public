## リスクマネジメント

放置するものではなく管理するものである。

* 回避・低減・移転・受容する
* 完全に排除できない
* 完全に予測できない
* ある程度定量化できる
* 主観で変わる

リスクを恐れて何もしないことが一番リスクが高いので判断に時間をかけすぎずに決断する。
